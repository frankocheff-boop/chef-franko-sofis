# Chef Franko Sofis

Sitio estático básico para el proyecto **chef-franko-sofis**.

## Qué contiene
- `index.html` : página principal.
- `styles.css` : estilos básicos.
- `.github/workflows/deploy.yml` : workflow que despliega a GitHub Pages.

## Cómo probarlo localmente
Abre `index.html` en tu navegador (doble click) o usa un servidor local (ej. Live Server en VS Code).

## Despliegue
Al hacer push a `main`, el workflow publicará el contenido en la rama `gh-pages` usando GitHub Actions. Activa GitHub Pages en la configuración del repositorio si quieres servir desde la rama `gh-pages`.
